Usage Tips
Checkbox and extension type (Chrome)
  • WHAT: The checkbox looks and functions differently for different extension types
  • DETAIL: Apps = checkbox has a red border; Themes = checkbox is disabled
Keyboard navigation in the extension popup
  • HOW: Trigger the popup then use arrow keys to navigate
  • CONTROLS: ↑, ↓ = Run through the extension list; Space = toggle enable/disable; o = Bring up extension's option page
Assign a keyboard shortcut to the extension
  • HOW: Go to the Chrome's Extensions settings page, scroll to the end and click "Configure commands"
Reason For Permission Requests
As the Chrome Web Store shows, SERP Extension Manager can:

Manage your apps, extensions, and themes
  • WHY: Without this permission the extension cannot perform its basic functions
